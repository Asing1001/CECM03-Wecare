# CECM03-Wecare

2014 資策會CECM03畢業專題 - Wecare

[Video introduction](https://www.youtube.com/watch?v=Orp-bMqC-FQ&list=PL_9-NlCIQkasCw_dKM3uKV-M04BZ6Pt7m&index=5&t=470s)

## 功能

### 平台管理系統

- 醫務管理
- 各項排程及行事曆管理
- 簡訊/E-mail 關懷提醒

### 網站

- **醫護人員**：
  - 工作行事曆
  - 報表分析

- **病患**：
  - 個人行事曆
  - 病歷時間軸
  - 檢查報告
  - 就診滿意度評價
  - 線上諮詢

- **健康購物**：
  - 銷售物品上下架
  - 訂單管理
  - 線上購物平台

### 市集

- BMI 計算器
- 醫生評價查詢

### 手機

- 特殊用藥提醒
